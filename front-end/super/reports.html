📍 Why is revenue calculated in INR?
The platform base currency is now Indian Rupee (₹) to support localized settlement for travel guides and domestic bookings. Exchange rates for USD/EUR are updated every 6 hours via the Currency API.
⚖️ How is 'Conversion Rate' derived?
It is the ratio of completed payments to the number of unique "Create Itinerary" sessions started within a 30-day period. Currently, we are seeing a 15% increase in mobile-initiated bookings.
👤 User Growth Insights
Most new users are signing up for domestic trips (Goa, Kerala, Rajasthan). We recommend increasing the travel guide pool in these regions by 10% next quarter.
🚀 System Reliability
API latency has decreased by 40ms following the implementation of the new caching layer for Flight searches. External error rates are below 0.1%.
