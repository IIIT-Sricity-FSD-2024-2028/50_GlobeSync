GlobeSync — Figma Design README
Overview
GlobeSync is a travel itinerary and coordination platform. The Figma file contains designs and workflows for all user roles: Public, Traveler, Guide, CustomerCare, and Admin.

  Goals
- Provide clear, role‑based workflows.
- Ensure trustworthy UI for bookings and payments.
- Deliver scalable components for dashboards, modals, and forms.

Structure
- Public: Home, Login, Info pages
- Traveler: Dashboard, Itinerary, Bookings, Expenses, Messages
- Guide: Dashboard, Suggestions, Chat
- CustomerCare: Support Dashboard, Tickets, Refunds
- Admin: Dashboard, User Management, Rules, Reports, Tokens

  Workflow
- Public → Login → Role Dashboards
- Traveler → Create Itinerary → Payment → Bookings
- Guide → Suggest Itinerary → Traveler Messages
- CustomerCare → Ticket Handling → Refunds
- Admin → User Management → Reports → Rules Config
